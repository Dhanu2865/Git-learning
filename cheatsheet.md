# Git Cheatsheet

### Setup
- git init
- git copy url

### Daily commands
- git status
- git add .
- git commit -m message
- git push origin main
- git pull

### Branching
- git branch
- git checkout -b branchname
- git checkout branchname

### Merge
- git merge branchname

### Logs
- git logs
- git logs -- oneline

## Undo
- git reset -- soft HEAD~1
- git revert <commit id>

### Stash
- git stash
- git stash pop
