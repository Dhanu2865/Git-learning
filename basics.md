# Git Basics
### What is version control?
Version control is used to track the changes in files over time.

### Types
- Centralized (SVN)
- Distributed (Git)

### Git Architecture
1. Working Directory - Your Files
2. Staging Area - Selected Changes
3. Repository - Saved Commits

## Flow:
Working -> Staging -> Commit

### File Lifecycle
- Untracked
- Modified
- Staged
- Commited
