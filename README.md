# Git-learning
### What is Git?
Git is a distributed version control used to track the changes in the code and lets multiple developers to have a full copy of the project.
### Why git?
- Tracks code history
- Collaborate with multiple developers
- Revert to previous versions
- Safer execution using branches
### Key Concepts
- Repository - The project which git is tracking
- Commit - Snapshot of changes
- Branch - Independent line of development
- Merge - Combine branches
- Remote - Online Repo
### Git Workflow
1. Clone Repository
2. Create a branch
3. Make Changes
4. Commit Change
5. Push branch
6. Create Pull Request
7. Merge Changes
