# Git Workflow
### Step by step

1. Pull latest Code
   - git pull origin main
2. Create new branch
   - git checkout -b branch name
3. Make changes
4. Stage the changes
   - git add .
5. Commit changes
   - git commit -m "message"
6. Push branch
   - git push origin branch name
7. Create Pull request
8. Code review
9. Merge to main
### Tip
- Never work directly on main
- Always pull before starting
